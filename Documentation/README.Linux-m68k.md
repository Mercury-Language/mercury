Mercury on Linux / Motorola 68000
=================================

This file documents the port of Mercury to Linux on Motorola 68000 (m68k).
Mercury was ported to Linux/m68k thanks to Roman Hodek
<Roman.Hodek@informatik.uni-erlangen.de>.

> [!NOTE]
> The Linux/m68k port has not been tested since 1998. If you have an m68k
> machine running Linux and run Mercury successfully, then please let us know.
